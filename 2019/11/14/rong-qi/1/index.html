Docker常用命令

2019-11-14
  • 容器

创建 Docker 环境（default）

docker-machine create --driver virtualbox default

或者

docker-machine create -d virtualbox default

或者

创建网易源的 Docker 环境

docker-machine create --engine-registry-mirror=http://hub-mirror.c.163.com -d virtualbox default

删除 Docker 环境（default）

 docker-machine rm default

开启 Docker 环境（default）

docker-machine start default

关闭 Docker 环境（default）

docker-machine stop default

重启 Docker 环境（default）

docker-machine restart default

查看 Docker 环境列表

docker-machine ls

查看 Docker 环境版本

docker-machine version

查看 Docker 环境信息 （default）

docker-machine env default

进入 Docker 环境 （default）

docker-machine ssh default

退出 Docker 环境

exit

查看 Docker 版本

docker version

查看所有容器（容器列表）

docker ps -a

搜索要下载的软件（mysql）

docker search mysql

下载安装软件 （mysql）

docker pull mysql

配置MySQL

docker run  -p 3307:3306 -e MYSQL_ROOT_PASSWORD=root -d mysql

查看下载的软件

docker images

删除下载的软件(IMAGE ID)

docker rmi -f  IMAGE ID

打开容器终端 （mysql）

docker run -i -t  mysql

后台运行

docker run -itd image_name /bin/bash

进入后台运行的 docker 容器 （容器ID）

docker exec -it 容器ID /bin/bash

启动容器（容器ID）

docker start 容器ID

停止容器（容器ID）

docker stop 容器ID

重启容器（容器ID）

docker restart 容器ID

删除容器（容器ID）

docker rm -f 容器ID

导入容器

 docker import ******

导出容器（容器ID）

docker export 容器ID

运行Web应用（Image）

docker run -p 8080:8080 -t Image

查看端口 （容器ID）

docker port 容器ID
