提升编程效率-Sublime Text 3 激活

2020年04月18日
  • 前端

Sublime Text 3.2.1 -Build 3207

介绍

这款软件具有良好的UI风格，丰富的插件，全面的代码提示功能，使用起来可以提升编程效率。

下载

链接: https://pan.baidu.com/s/12sm7DUzfYDXOJvOMHrOrQg 提取码: afr7 （10.3MB）包含破解补丁

汉化

1、安装 Package Control

第一步：按住 Ctrl + ~ 键,调出 sublime 的控制台；

第二步：在调出的控制台输入如下指令

import urllib.request,os;pf='Package Control.sublime-package';ipp=sublime.installed_packages_path();urllib.request.install_opener(urllib.request.build_opener(urllib.request.ProxyHandler()));open(os.path.join(ipp,pf),'wb').write(urllib.request.urlopen('http://sublime.wbond.net/'+pf.replace(' ','%20')).read())

回车，稍等一会，因为在下载文件所以非常卡，不要动等它下完，如果在 Preferences 里有 Package Control 就说明安装成功了；

第三步：点击菜单 Preferences(N) —> Package Control

或者使用快捷键 Ctrl + Shift + P调出 Package Control，输入Install Package 选择并回车，查找ChineseLocalization插件，选择回车安装成功后会自动切换成中文简体，可以在帮助 —>Language 中设置其他语言。

输入ChineseLocalization 回车等待，安装完成会自动切换成中文界面，我这里已经安装，不做过多演示

激活

首先屏蔽hosts

找到 C:/Windows/System32/drivers/etc/hosts,编辑 hosts 加入

    127.0.0.1        license.sublimehq.com

打开下载的压缩包

下载的压缩包内有两个文件

st3_twitter_patch.exe 复制到 SublimeText 3 安装目录(根目录)

右键以管理员身份运行，

点击patch，获取激活码
